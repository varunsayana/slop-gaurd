# Security Policy

If you discover a security vulnerability within SlopGuard, please let us know immediately by opening a secure advisory on GitHub or emailing the maintainers. Do not open a public issue.

## Supported Versions
Only the latest major version is actively supported for security updates. 

Since SlopGuard runs entirely locally and makes no external network requests natively, the attack surface is generally limited to malicious codebase analysis (e.g. maliciously crafted Python files causing regex DOS during parsing). Please report any such findings.
